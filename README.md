# Lox
Following along with the excellent CraftingInterpreters.com, but in Erlang.




COMPILE
-------

 > rebar3 compile

RUN TESTS
---------

 > rebar3 eunit

RUN INTERACTIVE LOX SHELL (REPL)
--------------------------------

 > ./lox_shell.sh

RUN LOX PROGRAM IN ILOX
-----------------------

 > ./ilox.sh hello

or

 > ./ilox.sh hello.lox

Where "hello" is the name of the program (see \programs folder).
